We hack you so no one else can

Rez0 Security legally hacks websites to help secure them with the goal of finding vulnerabilities before an attacker does.

Our Services

Rez0 Security provides the following offensive security services:

  • Web Application Testing
  • API Testing
  • iOS Application Testing


Our Mission

Rez0 Security was founded in Lexington, KY by Joseph Thacker (rez0) with the goal of helping companies secure their websites and applications by hacking them to find their vulnerabilities. These vulnerabilities can then be fixed before an attacker can take advantage of them.
